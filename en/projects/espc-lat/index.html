espc.lat

My personal wiki: an 'about me' and the place where I document the projects I'm building.

What is it?

espc.lat is this site: my personal wiki. The idea is to have a single place —with an encyclopedia look— to tell who I am and, above all, to document every project I build, each with its own page and info.

How it’s built

  • Astro as a static site generator (fast and lightweight).
  • Content in Markdown: each project is a file, so growing it is trivial.
  • A Wikipedia-like style (articles, infobox, blue links) combined with a bento grid on the home page.
  • Bilingual (Spanish / English) and with light/dark mode.
  • Deployed on GitHub Pages with a custom domain.

What’s next

  • Add my first real projects.
  • Get the @espc.lat email.
  • Polish the logo and the home page.

This article is both the first entry of the wiki and a reminder that the site is never “finished”: it grows with me.
